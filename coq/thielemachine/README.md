# Thiele Machine Proofs

The main Thiele Machine proof library with comprehensive coverage.

## Files (98)

### Core Semantics
- **ThieleMachine.v** - Main machine definition
- **CoreSemantics.v** - Core semantic definitions
- **ThieleFoundations.v** - Foundation theorems

### Bell Inequality & Tsirelson
- **BellInequality.v** - Bell inequality proofs
- **BellCheck.v** - Bell constraint checking
- **TsirelsonBoundBridge.v** - Tsirelson bound connection
- **QuantumAdmissibilityTsirelson.v** - Quantum admissibility

### Isomorphisms
- **LogicIsomorphism.v** - Logic isomorphism proofs
- **ComputationIsomorphism.v** - Computation isomorphisms
- **PartitionDiscoveryIsomorphism.v** - Partition discovery

### Verification
- Located in `verification/` subdirectory
- Deliverables for various theorems
- Bridge proofs and checkpoints

## Purpose

This is the largest proof module, containing the detailed implementation and verification of the Thiele Machine across multiple domains.
