# Thiele Universal Proofs

Universal Turing Machine and CPU definitions for the Thiele Machine.

## Files (7)

- **TM.v** - Turing Machine definitions
- **CPU.v** - CPU definitions
- **UTM_Encode.v** - UTM encoding
- **UTM_Rules.v** - UTM transition rules
- **UTM_Program.v** - UTM program representation
- **UTM_CoreLemmas.v** - Core lemmas for UTM
- **ThieleUniversal.v** - Universal Thiele Machine

## Purpose

These files establish the computational universality of the Thiele Machine by showing it can simulate any Turing Machine.
