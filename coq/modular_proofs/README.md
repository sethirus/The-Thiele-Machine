# Modular Proofs

Modular proof components for encoding and simulation.

## Files (7)

- **CornerstoneThiele.v** - Cornerstone theorems
- **Encoding.v** - Encoding proofs
- **EncodingBounds.v** - Encoding bounds
- **Minsky.v** - Minsky machine simulation
- **Simulation.v** - General simulation proofs
- **TM_Basics.v** - Turing Machine basics
- **Thiele_Basics.v** - Thiele Machine basics

## Purpose

These files provide modular building blocks for the larger proof development.
