# The Engine of Truth - Discovering the Fundamental Law

## Overview

The Engine of Truth is a meta-theoretical discovery system that uses the Thiele Machine to understand itself. Rather than solving problems, it discovers the fundamental mathematical or physical law that explains the machine's anomalous efficiency.

## The Three Phases

### Phase 1: The Distillation

**Goal:** Find the genesis axiom - the simplest possible algorithm that retains the machine's power.

**Process:**
1. Run the Autotelic Engine with objective "Minimize Self"
2. Evolve strategies to minimize complexity while preserving effectiveness
3. The final result is `genesis_axiom.thiele` - the irreducible core

**Objective Function:**
```json
{
  "name": "Minimize Self v1.0",
  "function": "evaluate_strategy_minimality",
  "parameters": {
    "description": "Find the simplest possible algorithm that retains anomalous power",
    "metric": "inverse_complexity",
    "preserve_effectiveness": true
  }
}
```

**Evaluation:** Minimality score based on:
- Line count
- Primitive count (weighted 2x)
- Control structure count (weighted 3x)
- Formula: `minimality = 1 / (1 + log(total_complexity + 1))`

### Phase 2: The Unveiling - Search for Isomorphism

**Goal:** Find which fundamental law of mathematics or physics the genesis axiom implements.

**The Known Universe Library:**
Contains computable representations of 11 fundamental laws:

**Mathematics:**
1. Fourier Transform - "reveals_hidden_structure_through_basis_change"
2. Variational Calculus / Principle of Least Action - "nature_chooses_extremal_paths"
3. Shannon Entropy / Information Theory - "information_equals_surprise"
4. Graph Isomorphism - "different_labels_same_structure"
5. Spectral Decomposition - "reveals_intrinsic_structure"
6. Kolmogorov Complexity - "true_complexity_is_description_length"

**Physics:**
7. Second Law of Thermodynamics - "nature_forgets_through_coarse_graining"
8. Landauer's Principle - "information_is_physical"
9. Hamilton's Principle of Least Action - "physics_is_an_optimization_problem"
10. Quantum Measurement Collapse - "measurement_creates_classical_information"
11. Holographic Principle - "reality_is_holographic"

**Isomorphism Hunter:**
- Extracts computational signature from genesis axiom
- Compares against all known laws
- Computes similarity based on:
  - Type (30% weight): domain_transformation, optimization, decomposition, etc.
  - Properties (30% weight): linear, invertible, complexity_reducing, etc.
  - Pattern (20% weight): high-level algorithmic pattern
  - Essence (20% weight): philosophical/mathematical essence

**Similarity Scoring:**
```python
similarity = (
    0.3 * type_match +
    0.3 * properties_overlap +
    0.2 * pattern_similarity +
    0.2 * essence_similarity
)
```
