How It Works

1. Create a Receipt: Drop your files into the generator. It computes cryptographic hashes (SHA-256) and creates a receipt JSON file.

2. Distribute: Share the receipt alongside your software. Anyone can verify it independently.

3. Verify: Drop the receipt into the verifier. It checks the cryptographic hash chain and optional signatures.

Learn more: Read the Receipt Guide or check the TRS-1.0 Specification
